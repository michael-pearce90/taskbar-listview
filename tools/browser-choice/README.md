# Browser Choice Tools

Browser Choice Tools is a proposed future module for TechTools for Windows.
There is no code, runtime, installer, redirector, service, browser extension,
or packaged release for this module.

The proposal is to explore transparent browser choice enforcement for
technicians and users who want local settings to respect the browser and
search provider they selected.

## Proposal Boundaries

Any future design must be:

- opt-in only;
- transparent and reversible;
- limited to local logging only;
- respectful of the user-selected browser and search provider;
- clear that Edge is not removed; and
- framed as browser choice enforcement, not Edge destruction.

Any future design must not:

- patch Microsoft binaries;
- replace system files;
- spoof region;
- disable security features;
- collect credentials, cookies, tokens, or sessions;
- claim that Edge is removed; or
- include browser redirect code until a reviewed module design is accepted.

## Current Status

This directory is a proposal placeholder only. It exists so the suite can name
the module idea and its safety boundary without implying implementation.
