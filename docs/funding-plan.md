# Funding Plan

This repository does not currently contain real funding handles or donation
URLs.

Do not create `.github/FUNDING.yml` until there is a real GitHub Sponsors,
Ko-fi, Buy Me a Coffee, Patreon, Liberapay, Open Collective, or custom funding
URL to publish. Do not invent handles or placeholder links.

## Options to Consider Later

- GitHub Sponsors for repository-native sponsorship.
- A fiscal host or collective if the project needs shared governance.
- A simple custom donations page if maintainers have a real published URL.
- No funding file if the project remains unfunded.

Any future funding text should explain what donations support, avoid implying
paid support for unsupported Windows builds, and keep module safety claims
separate from funding.
